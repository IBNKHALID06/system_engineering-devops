# 0x05. Processes and signals

This project covers the basics of Linux processes and signals, including how to manage process lifecycles, monitor running processes, and use signals to control process behavior.

## Tasks

0. **0-what-is-my-pid**: Displays the script's own Process ID (PID).
1. **1-list_your_processes**: Displays a list of currently running processes.
2. **2-show_your_bash_pid**: Displays lines containing the word "bash" from the process list.
3. **3-show_your_bash_pid_made_easy**: Displays the PID and process name of processes containing "bash".
4. **4-to_infinity_and_beyond**: Displays "To infinity and beyond" indefinitely with a 2-second sleep.
5. **5-dont_stop_me_now**: Stops the `4-to_infinity_and_beyond` process using `kill`.
6. **6-stop_me_if_you_can**: Stops the `4-to_infinity_and_beyond` process without using `kill` or `killall`.
7. **7-highlander**: Displays "To infinity and beyond" indefinitely and catches SIGTERM to print "I am invincible!!!".
   **67-stop_me_if_you_can**: Stops the `7-highlander` process.
8. **8-beheaded_process**: Kills the `7-highlander` process using SIGKILL.
